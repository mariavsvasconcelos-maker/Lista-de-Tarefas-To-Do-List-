✅ To-Do List | Vanilla JS & DOM Dinâmico

> Uma aplicação web minimalista e rápida para gerenciamento de tarefas, construída inteiramente "do zero" utilizando apenas JavaScript puro.

## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de consolidar os fundamentos do **JavaScript (Vanilla JS)**. O grande diferencial técnico desta aplicação é que o arquivo `index.html` atua apenas como um contêiner vazio. Toda a interface visual, elementos interativos e estilos básicos são gerados dinamicamente em tempo real através de **manipulação avançada do DOM**.

Além disso, a aplicação garante uma excelente experiência ao usuário salvando os dados localmente, o que significa que suas tarefas não são perdidas ao fechar o navegador.

## 🚀 Funcionalidades Principais

* **Criação de Tarefas:** Adicione novas pendências à sua lista instantaneamente.
* **Exclusão Interativa:** Remova tarefas concluídas ou canceladas com um simples clique.
* **Persistência de Dados (Save Game):** Integração com a API de `LocalStorage` do navegador para manter o estado da aplicação (suas tarefas continuam lá mesmo após recarregar a página).
* **Prevenção de Erros:** Validação de input que impede a criação de tarefas vazias.

## 🛠️ Tecnologias e Conceitos Aplicados

* **JavaScript (ES6+):** Motor principal da aplicação.
* **DOM (Document Object Model):** Uso de métodos como `document.createElement`, `appendChild` e `classList` para desenhar a tela via código.
* **JSON:** Conversão de dados (`JSON.stringify` e `JSON.parse`) para comunicação com o armazenamento local.
* **HTML5:** Estrutura base da página web.
